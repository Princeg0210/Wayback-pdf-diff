Wayback PDF Diff

A lightweight tool to compare two PDF documents and highlight differences in a clear, readable format.
Built as a prototype to extend the Internet Archive Wayback Machine’s diff capability to support PDFs.

⸻

🚀 Features
	•	Extracts text from PDF documents using PyMuPDF
	•	Cleans and normalizes text for accurate comparison
	•	Sentence-level + word-level diff visualization
	•	HTML-based side-by-side comparison view
	•	Simple web interface for uploading and comparing PDFs

⸻

🛠️ Tech Stack
	•	Python
	•	Flask
	•	PyMuPDF (fitz)
	•	difflib
